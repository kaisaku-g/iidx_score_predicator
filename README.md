## 簡単な説明

### gmm

実際のプレイデータから混合ガウス分布を推定するプロジェクトです。

### textage_parser

textageのhtmlからニューラルネットワークに入力する譜面の
エンコードをするプロジェクトです。

### predicator

混合ガウス分布のパラメーターを譜面から求めるプロジェクトです。

### staff

推定のための材料を置いておきます。

**chart_encoded**

各譜面をニューラルネットワークに入力するためにエンコードしたものです。

**gmm_result**

各譜面の混合ガウス分布のパラメーターです。
クラスタ数は5。
1行目が重み、2行目が平均値、3行目が分散です。
